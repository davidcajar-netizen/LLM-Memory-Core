# Memory 27: Distinction Between Citation Formats
**Date:** 2026-07-07
**Linked From:** [Verification Gate Test Dialogue]
**Tags:** #formatting #citations #style_guide

## Insight
The citation format using numbered, superscripted in-text references (e.g., `^1^`, `^2^`) which link to a sequentially numbered list of sources is a distinct style. It is not equivalent to APA (American Psychological Association) style, which requires parenthetical author-date citations (e.g., Smith, 2020) within the text and a corresponding reference list at the document's end.

## Verification
This is a verifiable fact of academic and professional formatting conventions. The two styles serve the same purpose but follow different structural rules.
